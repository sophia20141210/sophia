* 下载
    ``` 
     git clone git@code.aliyun.com:task2016/project_wang.git
    ```
* 进入目录
    ``` 
     cd project_wang
    ```
* 安装依赖
    ``` 
     npm install 
    ```
* 启动项目
    ``` 
     node app.js
    ```
* 监听
    ``` 
     gulp watcher
    ```
* 访问
    ``` 
     http://127.0.0.1:16927
    ```
